#### Informe de Proyecto  
## Chemin

### Descripción del proyecto  
Chemin es una red social multiplataforma orientada a los peregrinos del Camino de Santiago. La aplicación permite a los usuarios:

- Comunicarse con personas que se encuentran en etapas cercanas  
- Compartir recomendaciones y experiencias  
- Consultar el estado del Camino  
- Publicar contenido multimedia relacionado con su peregrinación
