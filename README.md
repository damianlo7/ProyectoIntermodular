#### Informe de Proyecto
# Chemin

## Descripción del proyecto
Chemin es una red social multiplataforma orientada a los peregrinos del Camino de Santiago. La aplicación permite comunicarse con personas que se encuentran en etapas cercanas, compartir recomendaciones, consultar el estado del Camino y publicar contenido multimedia sobre su experiencia.
